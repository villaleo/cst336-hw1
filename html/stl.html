The C++ STL

The Standard Template Library (STL)

The STL is a collection of template classes to provide programmers common functions and implementations to data structures. It's a library with iterators, containers, and algorithms. The only prerequisite to begin working with the STL in C++ is understanding template functions and classes, as most functions and every container are parameterized with template parameters.

The C++ STL's main components are:

  • Algorithms
  • Functions
  • Containers
  • Iterators
All of these functions and tools live in the std namespace.

Example: The std::pair<> class

std::pair<> encapsulates two values which may be of different types. These values are accessed via the first and second public members.

This object is used in the STL to represent a pair of values. It's used internally by the std::map<> container to store key-value pairs. std::pair<> is actually a specialization of the std::tuple<> class, a tuple of two elements.

On the example to the right, we have a std::pair<> object that contains a string and an integer, person. This object is used to store a person's name and age. After executing the code, the following output is printed:
